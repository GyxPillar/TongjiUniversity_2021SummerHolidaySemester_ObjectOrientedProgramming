#### Introduction:####


实现了分数类的某些基本功能及其测试：

1.单目运算符
 
	Inverse     取反

	Reciprocal  求倒数

	Reduction    约分

	DtoF 小数转分数

	StoF 字符串转分数

2.双目运算符

	＋ - * / 

 	< > <= >=  == ！=

3.输出运算符重载



----------

#### Files: ####



头文件
	
	文件名:fraction.h
	功能  :实现了分数类fraction的声明，完成了部分宏定义#define与类型定义typedef。
源文件

	文件名:fraction_main.cpp
	功能  :进行分数类各功能的测试。

	文件名:fraction_menu.cpp
	功能  :在进行分数类功能测试时，展示菜单并控制测试模式。

	文件名:fraction_unary_operator.cpp
	功能  :实现分数类单目运算符重载。

	文件名:fraction_binary_operator.cpp
	功能  :实现分数类双目运算符重载。

	文件名:fraction_out.cpp
	功能  :实现分数类输出运算符的重载。
	
	文件名:fraction_test_mode_1.cpp
	功能  :实现分数类功能的测试1模式

	文件名:fraction_test_mode_2.cpp
	功能  :实现分数类功能的测试2模式


----------

#### Usages: ####

	1.测试共两个模式，供用户自身选择:
	模式一无需输入，测试数据由程序本身提供，输出数据于控制台窗口和文件.
	模式二需要用户选择测试功能、输入测试数据，输出数据于控制台窗口和文件.

	2.请保证输入数据类型及范围正确.(仅保证了异常操作处理，如除以0，对0取倒数;未对输入数据作异常处理)

	3.对于错误操作(如除以0、对0取倒数等)，进行特判，并将分数置为 0/1 ，所以请尽量避免输入 0/1 作为数据0.0

	4.由于3.，默认将0.0记作 0/2 以区分.
